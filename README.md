# Ledger Management System

## 📌 Description
A console-based Ledger Management System developed using Core Java to manage financial transactions efficiently. The application implements Object-Oriented Programming (OOP) concepts, data structures, and file handling for real-world financial tracking.

## 🚀 Features
- User Authentication (Signup, Login, Password Validation)
- Add, Edit, Delete Transactions
- Filter Transactions by Type and Month
- Category-wise Spending Analysis
- Budget Management with Alerts
- Search and Sort Functionality
- Export Statement using File Handling

## 🛠 Tech Stack
- Java (Core Java, OOP)
- Data Structures (ArrayList, HashMap)
- File Handling
- Exception Handling

## ▶️ How to Run
1. Compile the program
2. Run LedgerManager.java
3. Follow menu-driven interface
