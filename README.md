# Ledger Management System

A console-based Java application to manage personal financial transactions efficiently.

## Features
- User Authentication (Signup/Login/Forgot Password)
- Add, Edit, Delete Transactions
- Filter by Type and Month
- Category-wise Spending Analysis
- Budget Management with Alerts
- Search and Sort Transactions
- Export Statement to File

## Tech Stack
- Java (Core Java, OOP)
- Data Structures (ArrayList, HashMap)
- File Handling
- Developed using IntelliJ IDEA and Notepad++
